Help

Gemoto is a data science tool that's designed to let you see how emotions vary geographically.

To get started, select an emotion to view on the map (or select several to overlay together). You can pan around to explore different locations, or use the search bar to jump to a specific address. Try seeing how people are feeling around your own home!

About

This tool was created as a senior capstone project for a group of computer science majors at the University of Tulsa.

Methodology

Gemoto runs by polling Twitter for recent geo-tagged tweets and sending them to IBM's Watson Tone Analyzer API. Watson analyzes each tweet and looks for a range of emotions, like joy, anger, sadness, and fear. If Watson is confident that a tweet conveys a particular emotion, it will flag the tweet as such.

Once all the tweets have been analyzed, the data is packaged and displayed here using Mapbox, a geographic visualization tool which powers services like Foursquare, Lonely Planet, Facebook, the Financial Times, The Weather Channel, and Snapchat. You can use the selectors here to manipulate which emotions are overlayed on the map and pan around to see different areas.

Currently, only data for the nearby Oklahoma area where Gemoto was developed is available. In the future, data will be updated periodically to adjust as emotions change over time, and will be collected from more parts of the world.
